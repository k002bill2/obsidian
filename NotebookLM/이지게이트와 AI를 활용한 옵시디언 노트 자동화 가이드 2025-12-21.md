# 이지게이트와 AI를 활용한 옵시디언 노트 자동화 가이드

제공해주신 소스들은 옵시디언(Obsidian)에서 웹 브라우저처럼 외부 사이트를 활용하고 AI로 노트를 자동 생성할 수 있게 해주는 **'이지게이트(EasyGate)' 플러그인**과 AI 기반 구글 폼 생성 도구인 **'티보드 폼(T-Board Form)'**에 대한 상세 가이드를 담고 있습니다. 주요 내용을 이해하기 쉽게 정리해 드립니다.

## 1. 이지게이트(EasyGate) 플러그인 소개 및 주요 기능

이지게이트는 기존 '오픈 게이트' 플러그인의 불편함을 개선하여 제작된 **커스텀 플러그인**으로, 옵시디언 내부에서 웹 브라우저를 사용하는 것과 같은 경험을 제공합니다.
• **웹 브라우저 인터페이스:** 옵시디언 창 안에서 앞/뒤 페이지 이동 버튼과 X(닫기) 버튼을 제공하며, 여러 사이트를 **탭 형식**으로 띄워두고 자유롭게 이동할 수 있습니다.
• **콘텐츠 삽입 기능:** 웹 페이지의 내용을 선택하여 현재 커서 위치, 노트의 가장 아래, 혹은 **새로운 노트**로 즉시 만들 수 있는 버튼을 지원합니다.
• **AI API 연동:** Gemini, Claude, OpenAI 등의 API를 입력하여 웹 콘텐츠를 AI로 가공(요약, Q&A 변환 등)한 뒤 노트화할 수 있습니다. 이를 통해 복잡한 프롬프트 입력 없이도 텍스트를 마크다운 형식이나 표 형식으로 변환하여 저장할 수 있습니다.
• **자동 프로퍼티스(Properties) 생성:** 노트를 생성할 때 출처 URL과 같은 기본 정보가 자동으로 포함되어 관리가 용이합니다.

## 2. AI를 활용한 노트 생성 예시

• **Gemini 활용:** 이지게이트 내에서 Gemini를 열고 정보를 검색한 뒤, 필요한 부분을 드래그하여 AI 분석 모달창에서 'Q&A 형식'이나 '표 형식'으로 변환하여 새 노트로 저장할 수 있습니다.
• **NotebookLM 활용:** 여행 계획과 같은 긴 정보를 검색한 후, 전체 글을 복사하여 AI 템플릿(기본 요약 등)을 적용해 옵시디언 노트를 순식간에 완성할 수 있습니다.

## 3. 티보드 폼(T-Board Form)과 옵시디언 연동

AI를 이용해 **구글 설문지나 퀴즈를 자동으로 생성**해주는 도구입니다.
• **자동 생성:** 주제, 목적, 응답자 대상 등을 입력하면 AI가 문체를 조절하여 구글 폼을 만들어주며, 결과물은 사용자의 구글 드라이브에 저장됩니다.
• **Gemini API 필요:** 텍스트 위주의 생성은 비용 부담이 적으며(약 3,000원 수준), 결제 카드가 연결된 티어 1 계정을 사용하면 제한 없이 활용 가능합니다.
• **옵시디언 임베드:** 생성된 구글 폼의 '인베드 HTML' 코드를 복사하여 옵시디언 노트에 붙여넣으면, **옵시디언 안에서 바로 설문 응답 및 제출**이 가능해집니다.

## 4. 설치 및 설정 방법

이지게이트는 정식 커뮤니티 플러그인이 아닌 베타 버전이므로 **BRAT 플러그인**을 통해 설치해야 합니다.
1. **BRAT 설치:** 옵시디언 설정 > 커뮤니티 플러그인에서 'BRAT'을 검색하여 설치 및 활성화합니다.
2. **이지게이트 등록:** BRAT 설정에서 'Add Beta plugin'을 선택하고 공유받은 링크를 입력합니다.
3. **초기 설정:** 첫 번째 게이트(예: Gemini 주소)를 설정하고, 단축키(`Ctrl/Cmd + Shift + G`)를 지정하여 빠르게 불러올 수 있도록 합니다.

--------------------------------------------------------------------------------

**이해를 돕기 위한 비유:** 이지게이트는 옵시디언이라는 서재 안에 **'마법의 창문'**을 다는 것과 같습니다. 이 창문을 통해 밖(웹사이트)을 내다볼 수 있을 뿐만 아니라, 창밖에 지나가는 정보(AI 검색 결과)를 마법 지팡이(AI 기능)로 톡 건드려 즉시 정리된 문서로 만들어 서재 책꽂이(노트)에 꽂아두는 것과 같은 편리함을 제공합니다.
