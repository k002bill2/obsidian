# Claude Code 업데이트: 지능형 브라우저 제어와 워크플로우 혁신

제시해주신 소스 내용을 바탕으로, Claude Code의 최신 업데이트 기능과 세부적인 작동 방식을 더욱 상세하게 보강하여 정리해 드립니다.
1. 강력해진 브라우저 제어 및 통합 (Claude in Chrome)
Claude Code는 단순한 코드 작성 도구를 넘어, 브라우저를 직접 제어하고 웹 환경과 상호작용하는 능력을 갖추게 되었습니다.
• 브라우저 확장 프로그램 연결: cloud update를 통해 최신 버전으로 업데이트한 후, /chrome 명령어로 확장 프로그램의 설치 및 활성화 상태를 관리할 수 있습니다1.
• 자율적인 웹 작업 수행: Claude Code는 새로운 탭을 생성하고, 웹페이지 내에서 텍스트 입력, 버튼 클릭, 스크롤링 등의 작업을 수행합니다1.... 특히 스크린샷을 찍어 페이지의 레이아웃을 시각적으로 이해할 수 있는 기능이 포함되어 있어, ChatGPT와 같은 외부 도구에 접속해 질문을 던지고 답변을 가져오는 복잡한 워크플로우도 소화 가능합니다12.
• 실시간 디버깅 및 콘솔 접근: 브라우저 내부의 콘솔 로그(Console logs)를 직접 읽을 수 있습니다24. 이를 통해 웹 인터페이스에서 작업 중 오류가 발생하면 Claude Code가 스스로 로그를 분석하여 문제를 해결(self-debug)할 수 있습니다2.
2. 생산성 극대화를 위한 인터페이스 개선
사용자가 코딩에 더 집중할 수 있도록 시각적 요소와 편의 기능이 대폭 강화되었습니다.
• 코드 변경 사항 가시성 향상: 코드베이스의 변경 내용을 보여주는 구문 강조(Syntax Highlighting) 기능이 개선되었습니다56. 새로운 컬러 스키마와 강조된 행 번호를 통해 이전 버전보다 훨씬 명확하게 수정된 부분을 파악할 수 있습니다6.
• 지능형 프롬프트 제안: 사용자의 작업을 감지하여 다음 행동을 추천합니다5. 예를 들어 로컬에서 커밋을 완료하면 자동으로 "push"를 제안하며, 사용자는 Tab 키를 눌러 이를 즉시 실행할 수 있습니다5. (이 기능이 번거롭다면 /config에서 끌 수 있습니다)5.
• 정밀한 사용량 모니터링: /status line 명령에 통화(currency) 기반 사용량 필드가 추가되었습니다67. 이를 통해 현재 컨텍스트 창에서 사용된 비용이나 데이터양을 실시간으로 더 정확하게 확인할 수 있습니다7.
3. 단축키 체계 변경 및 모델 관리
효율적인 조작을 위해 주요 단축키가 변경되었으며, 모델 전환이 더 유연해졌습니다.
• 사고 모드(Thinking Mode) 전환: 기존 Tab 키에서 **Alt+T (Windows) 또는 Option+T (Mac)**로 변경되었습니다4.
    ◦ Mac 사용자 주의사항: Mac에서 Option+T를 누를 때 특수문자(†)가 입력된다면, 터미널 설정(Warp 등)에서 'Left Option key is Meta' 옵션을 활성화해야 정상 작동합니다45.
• 실시간 모델 교체: 프롬프트를 작성하는 도중에도 **Alt+P (Windows) 또는 Option+P (Mac)**를 눌러 즉시 다른 모델로 바꿀 수 있습니다7. 이전처럼 입력을 취소하고 모델을 바꾼 뒤 다시 입력할 필요가 없어 매우 편리합니다7.
• 기본 설정 변경: 이제 Opus 4.5 모델을 사용할 때는 사고 모드가 기본적으로 활성화된 상태로 시작됩니다7.
4. 확장된 생태계 및 기타 기능
• 공식 플러그인 마켓플레이스: /plugins 명령어를 통해 Anthropic이 제공하는 공식 플러그인 목록을 확인할 수 있습니다6. 여기에는 Chrome 개발자 도구, 코드 리뷰 도구, 마이그레이션 도구 등이 포함되어 있으며, 자동 업데이트를 비활성화하여 특정 버전을 유지할 수도 있습니다6.
• 모바일 연동 및 단축키 삭제: /mobile 명령어로 모바일 앱 다운로드 QR 코드를 볼 수 있으며, 기존에 cloud.md 파일에 내용을 추가할 때 쓰던 해시태그(#) 단축키는 더 이상 지원되지 않습니다47.
Claude Code의 이러한 발전은 경쟁 모델을 모방하기보다 내부 개발자와 사용자들이 실제로 필요로 하는 기능을 우선적으로 구축한다는 개발 철학에서 비롯되었습니다8.
이러한 업데이트를 한 문장으로 비유하자면, **"Claude Code가 이제는 책상 앞에 앉아 코딩만 하는 조수를 넘어, 필요할 때 직접 밖으로 나가(브라우저) 자료를 찾고 도구(플러그인)를 챙겨오며 실시간으로 비용까지 정산해주는 만능 비서로 진화했다"**고 볼 수 있습니다.
