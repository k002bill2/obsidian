# 미드저니 나만의 개인 작업실 구축 가이드

제공해주신 소스 내용을 바탕으로 디스코드 개인 서버 생성 방법과 미드저니 설정 및 주요 기능에 대해 상세히 정리해 드립니다.
1. 디스코드 개인 서버 만들기 및 관리
본인의 이미지만 따로 모아 효율적으로 관리하기 위해 개인 서버를 만드는 과정은 다음과 같습니다.
• 서버 생성 단계:
    1. 디스코드 왼쪽의 **플러스(+) 아이콘(서버 추가하기)**을 클릭합니다1.
    2. **'직접 만들기'**를 선택한 후, **'나와 친구들을 위한 서버'**를 클릭합니다1.
    3. 원하는 서버 이름(예: 전용 미드저니)을 입력하고 **'만들기'**를 누르면 생성이 완료됩니다1.
• 미드저니 봇(Midjourney Bot) 초대:
    1. 미드저니 공식 서버로 이동하여 오른쪽 멤버 목록에서 **'Midjourney Bot'**을 찾아 클릭합니다1.
    2. '앱 추가' 버튼을 누른 후 **'서버에 추가'**를 선택합니다1.
    3. 조금 전 만든 개인 서버를 선택하고 '계속하기'와 **'승인'**을 차례로 누르면 개인 서버에서 미드저니를 사용할 준비가 끝납니다1.
• 서버 삭제 방법: 만약 서버를 지우고 싶다면 서버 아이콘 우클릭 > 서버 설정 > 일반 메뉴에서 가장 아래에 있는 **'서버 삭제'**를 클릭하고 서버 이름을 입력하면 됩니다1.
2. 미드저니 설정(/settings) 상세 내용
채팅창에 /settings를 입력하면 미드저니의 다양한 기능을 설정할 수 있는 창이 나타납니다1.
• 버전 선택: 현재 기본값인 Model V6를 포함하여 V1부터 V6까지 선택 가능합니다. 최신 버전일수록 결과물의 퀄리티가 훨씬 좋습니다2.
• 스타일라이즈(Stylize) 모드: 스타일 적용 강도를 Low, Med, High, Very High 4단계로 조절할 수 있으며, 일반적으로는 기본값을 유지합니다2.
• 공개 및 스텔스 모드: 기본은 공개 모드(Public)이며, 프로 플랜(Pro Plan) 이상 사용자는 결과물을 남들에게 숨길 수 있는 스텔스(Stealth) 모드를 사용할 수 있습니다2.
• 속도 모드: 플랜에 따라 제공되는 패스트(Fast) 모드 시간 내에서 생성 속도를 조절할 수 있습니다2.
• 리믹스(Remix) 모드: 기존 이미지의 일부를 수정하거나 새로운 요소를 더할 때 사용하는 유용한 기능입니다2.
• 기타 옵션: 베리에이션(Variation) 모드를 통해 생성된 이미지를 더 다양하게 변화시킬 수 있으며, 로우(Raw) 모드는 버전 5.1 이후부터 적용 가능한 옵션입니다2.
3. 사용자 정보 확인(/info)
자신의 계정 상태가 궁금할 때는 /info 명령어를 사용합니다. 이 명령어를 통해 현재 사용 중인 플랜, 남은 패스트 모드 시간, 지금까지 생성한 총 이미지 개수 등을 한눈에 확인할 수 있습니다2. 더 자세한 설정 차이나 내용은 미드저니 공식 문서 사이트에서 확인이 가능합니다2.
--------------------------------------------------------------------------------
비유하자면, 개인 서버를 만드는 것은 수많은 사람이 이용하는 공용 광장에서 벗어나 나만의 개인 작업실을 꾸리는 것과 같습니다. 이 작업실에 미드저니라는 **유능한 조수(봇)**를 초대하고, 설정 버튼을 통해 **조수의 작업 도구와 속도(버전 및 모드)**를 내 입맛에 맞게 조율하는 과정이라고 이해하시면 됩니다.
