

주요 명령어

	•	일일 보고서: `ccusage` 또는 `ccusage daily`
	•	월별 집계: `ccusage monthly`
	•	세션별 사용량: `ccusage session`
	•	5시간 청구 블록별 집계: `ccusage blocks`
	•	실시간 모니터링: `ccusage blocks --live`
	•	모델별 비용 상세: `ccusage daily --breakdown`
	•	기간 지정: `ccusage daily --since YYYYMMDD --until YYYYMMDD`
	•	JSON 출력: `ccusage daily --json`.
