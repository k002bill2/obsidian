# Chrome 다운로드 위치 설정 가이드

## NotebookLM-Import로 자동 저장 설정

### 방법 1: 항상 묻기 (추천 ⭐)

이 방법을 사용하면 매번 저장 위치를 선택할 수 있습니다.

1. **Chrome 설정 열기**
   - 주소창에 입력: `chrome://settings/downloads`
   - 또는: ⋮ (우측 상단) → 설정 → 다운로드

2. **"다운로드 전 각 파일의 저장 위치 확인" 켜기**
   ```
   ☑ 다운로드 전 각 파일의 저장 위치 확인
   ```

3. **이제 Export 할 때**
   - 파일 저장 대화상자가 나타남
   - 데스크톱의 "NotebookLM-Import 가상본" 선택
   - 저장!

### 방법 2: 기본 위치 변경

항상 NotebookLM-Import에 저장하려면:

1. **Chrome 설정 → 다운로드**
   - 주소창: `chrome://settings/downloads`

2. **위치 변경**
   - "위치" 옆의 [변경] 버튼 클릭
   - 폴더 선택: `/Users/younghwankang/NotebookLM-Import`
   - 확인

3. **완료!**
   - 이제 모든 다운로드가 NotebookLM-Import로 저장됨
   - (주의: 다른 파일도 여기 저장됨)

## 빠른 접근 방법

### Finder에서 즐겨찾기 추가

1. Finder 열기
2. `Shift + Cmd + G` 누르기
3. 입력: `~/NotebookLM-Import`
4. 폴더를 사이드바의 "즐겨찾기"로 드래그

이제 파일 저장 대화상자에서 클릭 한 번에 접근!

## 테스트

1. Chrome에서 아무 이미지나 우클릭 → "이미지를 다른 이름으로 저장"
2. NotebookLM-Import 폴더가 보이는지 확인
3. 저장 테스트

## 문제 해결

### 폴더가 보이지 않아요
- Finder에서 `~/NotebookLM-Import` 경로 확인
- 폴더가 존재하는지 확인:
  ```bash
  ls -la ~/NotebookLM-Import
  ```

### 자동 저장이 안 돼요
- Chrome 설정이 올바른지 재확인
- Chrome 재시작
